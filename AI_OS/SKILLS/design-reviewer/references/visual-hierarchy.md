# Visual Hierarchy & Consistency Evaluation

## Visual Hierarchy

### Size Hierarchy
- **Primary**: Largest elements (headlines, hero images, main CTAs)
- **Secondary**: Medium elements (subheadings, supporting content)
- **Tertiary**: Smallest elements (body text, metadata, fine print)

**Evaluation**:
- Is there clear size differentiation between hierarchy levels?
- Do the most important elements have the most visual weight?
- Is the size ratio between levels consistent (1.25-1.5x recommended)?

### Contrast & Color Hierarchy
- **High contrast**: Primary actions, critical information
- **Medium contrast**: Supporting content, secondary actions
- **Low contrast**: De-emphasized elements, disabled states

**Evaluation**:
- Does contrast guide the eye to important elements?
- Is color used consistently to indicate meaning?
- Does the design work in grayscale (squint test)?

### Spatial Hierarchy
- **Proximity**: Related items grouped together
- **Whitespace**: Important elements have room to breathe
- **Alignment**: Elements aligned to grid create order

**Evaluation**:
- Are related items clearly grouped?
- Is there adequate whitespace around key elements?
- Do elements align to a consistent grid?

---

## Consistency Checklist

- [ ] Buttons: Same style for same function
- [ ] Links: Consistent appearance and behavior
- [ ] Cards: Consistent structure, padding, shadows
- [ ] Icons: Same style family, consistent sizing
- [ ] Spacing: Predictable vertical rhythm
- [ ] Colors: Limited palette, consistent meaning
- [ ] Typography: Limited scale, consistent application

---

## Quick Hierarchy Tests

**Squint Test**: 
Blur your vision or step back from the screen. Can you still identify the most important elements? If everything blends together, hierarchy is weak.

**5-Second Test**:
Show the design for 5 seconds. Can viewers identify:
1. What the page is about?
2. What action they should take?
3. What's most important?

If not, hierarchy needs work.

**Scan Path Test**:
Where does your eye go first? Second? Third? Does this match the intended priority of information?

## Hierarchy Scorecard

| Dimension | Score (1-5) | Notes |
|-----------|-------------|-------|
| Size hierarchy | | Clear differentiation between levels |
| Contrast hierarchy | | Important elements stand out |
| Spatial hierarchy | | Grouping and whitespace support meaning |
| Consistency | | Same elements look and behave the same |
| Scan path | | Eye flow matches content priority |

**Scoring**:
- 4.5-5.0 = Crystal clear hierarchy
- 3.5-4.4 = Good, minor confusion points
- 2.5-3.4 = Hierarchy muddy in places
- Below 2.5 = No clear hierarchy, everything competing
