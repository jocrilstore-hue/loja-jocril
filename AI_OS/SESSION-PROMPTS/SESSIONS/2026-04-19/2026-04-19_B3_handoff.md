- `lib/types.ts` — replaced the mock `CartItem` (unused in code) with the real shape: `{ variantId, sku, productName, sizeName, quantity, unitPrice, totalPrice, imageUrl?, stockQuantity }`. Added `Cart` type.

- `app/(store)/layout.tsx` — dropped mock `cart={[{qty:3}]}` from StoreHeader prop call.
- `components/store/StoreHeader.tsx` — removed `cart` prop + inline `CartItem` type. Reads `useCart().cart.totalItems` directly.
- `app/(store)/carrinho/page.tsx` — full rewrite. Uses `useCart()`; EmptyCart state; qty +/−, remove, clear wired; step-1 "Continuar" disabled when empty. Cross-sell tiles now link to `/produtos`. Step 2/3 UI (address, shipping, payment, success) kept as mock.

Added `handleAdd` in PDPClient that builds the CartItem payload from `curSize + unit + qty` and calls `addToCart`. `InfoRail` accepts new `onAdd` prop and attaches it to the button (button shows "Pedir orçamento" + disabled when `unit == null`).

## Gotchas / deferred

1. **No toast system** — old project used shadcn `useToast()`. Not ported (no shadcn in new project). Add-to-cart has no visible confirmation beyond the header count incrementing. Flag for a minimal custom toast later.
2. **Material dropped from cart row** — cart item shape has no `material` field (old shape didn't either — old project displayed `material` via a supplementary query on the cart page, we chose not to port that). Cart row shows SKU + productName + sizeName.
3. **Stock label is a static string** on cart rows ("Em stock · envio 48h"). Not per-variant live check.
4. **Savings line removed** — old mock had `tierDiscount` per item; real cart items only carry the already-discounted `unitPrice`. Re-adding would require capturing pre-discount price at add time. Flag for B4.
5. **Cross-sell tiles still mock** in the cart page.
6. **`stockQuantity` snapshot issue** — at add-to-cart time, PDP passes the *current* variant's stock even when user selected a sibling size. Close enough for cart display; re-check at checkout.
7. **ShippingOptions / PaymentOptions / AddressBlock / SuccessPanel** all still mock — B4 territory.
